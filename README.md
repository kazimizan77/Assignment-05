- 1️⃣ What is the difference between var, let, and const?
  -Ans: JavaScript has three ways to declare variables: var, let, and const.
  
    -var is function scoped and it allows both re-declaration and updating the value.
    -let is block scoped and allows updating the value but does not allow re-declaration in the same block.
    -const is also block scoped, but its value cannot be reassigned after it is declared.

- 2️⃣ What is the spread operator (...)?
  -Ans: The spread operator (...) is used to expand elements of an array or object.It makes it easier to copy arrays, merge arrays or objects, and pass values into functions.
  It helps make the code shorter and cleaner.

- 3️⃣ What is the difference between map(), filter(), and forEach()?
  -Ans: map(), filter(), and forEach() are JavaScript array methods used to work with array elements.

    -map() creates a new array by transforming each element.
    -filter() creates a new array containing only the elements that match a specific condition.
    -forEach() loops through each element of an array and performs an action, but it does not return a new array.

- 4️⃣ What is an arrow function?
  -Ans: An arrow function is a shorter way to write functions in JavaScript. It was introduced in ES6 and uses the => syntax instead of the function keyword.

  Arrow functions make the code more concise and easier to read.

- 5️⃣ What are template literals?
  -Ans: Template literals are strings written using backticks (`) that allow variables and expressions to be embedded directly. Introduced in ES6, they make strings more readable and support easy interpolation and multi-line text without complex concatenation.

      -Use backticks (`) to embed variables and expressions directly in strings.
      -Introduced in ES6, they support easy interpolation and multi-line strings.
